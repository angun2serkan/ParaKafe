# PARA KAFE

## SUMMARY
* It is a Windows form application that i used C# programming language.
* An application for cafes to track their customers' transactions (I wrote codes and named everything in Turkish.).
* Each tables bill kept in its own page and we can add or delete products. 
* It is possible to change table number.  We can add new products to Cafe’s menu from “Ürünler” form by giving a name and cost to product. 
* All the transactions which made in this app are stored in “Geçmiþ Sipariþler” Form.
